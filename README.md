# GameBackLog

Simple web application that adds games to a list to track.

## Technologies

- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- EJS, Embedded JavaScript
- Morgan
- Mongoose
- Method-Override
- MongoDB
- Heroku
- Google Fonts
